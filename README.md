# E-Commerce Sales SQL Analysis

## Project Overview
This project demonstrates SQL-based financial data analysis using MySQL Workbench. The objective was to design a relational database and extract business insights from transactional sales data.

## Key Analysis Areas
- Total Revenue, Profit, and Units Sold
- Revenue by Product Category
- Profit by Region
- Monthly Revenue Trends
- Return on Ad Spend (ROAS)

## Tools Used
- MySQL Workbench
- SQL (Aggregate Functions, GROUP BY, ORDER BY)
- Microsoft Excel

## Skills Demonstrated
- Database design
- Data validation
- Business intelligence querying
- Financial performance analysis
- Structured problem-solving
